# SkillTrade – The Barter Network

SkillTrade – Full Stack Skill Exchange Platform

SkillTrade is a modern full-stack web application where users can exchange skills, send swap requests, chat with users, and manage notifications.

🚀 Tech Stack
React.js
FastAPI
MongoDB Atlas
Tailwind CSS
JWT Authentication
Vercel & Render Deployment

✨ Features
User Authentication
Skill Swap Requests
Messaging System
Notifications
Dashboard Analytics
Protected Routes
Responsive UI

## ⚙️ Local Setup

### Frontend
```bash
cd client
npm install
npm run dev
```

### Backend
```bash
cd python-backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```
